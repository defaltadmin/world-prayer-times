How to Use

1. The ruler shows your local time. The red NOW line marks the current moment.

2. Prayer blocks show the full window for each Salah — wider blocks reflect longer prayer windows (e.g. Fajr spans from Adhan to Sunrise).

3. Drag the cyan bar to select a meeting window. Resize by dragging the left/right edges. Arrow keys also move it.

4. The status badge turns red if your selection overlaps any prayer window, green if it's clear for everyone.

5. Click the 📖 icon to view Sannatayn course links and enroll in classes.

6. Use the 🕐 button to scroll the timeline back to the current time.

Privacy & Data

No data collected

This app does not collect, store, or transmit any personal data. No accounts, no analytics, no tracking.

What stays on your device

  • Selected cities (localStorage)
  • Theme, language, and time format preferences (localStorage)
  • Your approximate timezone (localStorage) — precise GPS coords are never stored
  • Class enrollment selections (localStorage)

External requests

  • api.aladhan.com — prayer time data (coordinates sent, no user ID)
  • nominatim.openstreetmap.org — city search only (text sent, no user ID)
  • fonts.googleapis.com — font loading

Geolocation

If you grant location permission, coordinates are used once to set your timezone. Only a rounded value (±11 km) is stored locally. You can revoke this in browser settings at any time.
